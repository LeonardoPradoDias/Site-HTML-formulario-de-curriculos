Sobre o projeto:

Este é um site em HTML e CSS para a disciplina Multimídia e Hipermídia. Consiste em uma plataforma em que o usuário preenche dados para conseguir uma vaga de emprego em uma empresa fictícia. Consiste de oito páginas HTML e uma folha de estilo em CSS. Os dados são salvos utilizando LocalStorage, e ficam salvos no browser.
